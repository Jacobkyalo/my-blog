---
title: How to install TailwindCSS in a Vite and React project
publishedAt: 2023-12-23
description: Learn how install TailwindCSS in an existing or new Vite and React project quickly and simple
---

## 1. Create a new Vite and React project using the following command

```bash
npx create-vite@latest my-react-app --template react
```

## 2. Change into the project directory

```bash
cd my-react-app
```

## 3. Install the required dependencies

```bash
npm install tailwindcss@latest postcss@latest autoprefixer@latest
```

## 4. Generate the TailwindCSS configuration file

```bash
npx tailwindcss init -p
```

## 5. Open the _tailwind.config.js_ file and paste the following code or customize the configuration as needed

```javascript
/** @type {import('tailwindcss').Config} */
export default {
  content: ["./index.html", "./src/**/*.{js,ts,jsx,tsx}"],
  theme: {
    extend: {},
  },
  plugins: [],
};
```

## 6. Create a _postcss.config.js_ file in the project root with the following content

```javascript
module.exports = {
  plugins: {
    tailwindcss: {},
    autoprefixer: {},
  },
};
```

## 7. Import the Tailwind CSS styles in your _index.css_ file

```css
@import "tailwindcss/base";
@import "tailwindcss/components";
@import "tailwindcss/utilities";
```

## 8. Import the _index.css_ file in your _index.js_ file

```javascript
import "./index.css";
```

## 9. Start using Tailwindâ€™s utility classes to style your content

```javascript
export default function App() {
  return <h1 className="text-3xl font-bold underline">Hello world!</h1>;
}
```

## 10. Start the development server

```bash
npm run dev
```

That's it! You have successfully installed Tailwind CSS in your Vite React project.
